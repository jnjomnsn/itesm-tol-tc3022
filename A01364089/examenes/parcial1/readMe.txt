Primer Examen Parcial

El objetivo del programa es dibujar un triangulo de Sierpinsky, basado en fractales de forma triangular y ponerlos dentros de las caras de una piramide con una base cuadrada para mostrarlo girando.

Se utilizo un algoritmo para calcular los puntos que van dividiendo en 2 al triangulo principal para de esta manera crear una forma recursiva de hacerlo, en este caso n = 5 el dibujo realiza 5 veces la division del triangulo principal y va pintando cada nueva division del mismo creando este fractal denominado como triangulo de Sierpinsky.

Se utilizaron los siguientes puntos para delimitar la cara del triangulo:

x1, y1   indican el primer punto del triangulo a dibujar
x2, y2   indican el segundo punto del triangulo a dibujar
x3,y3    indican el tercer punto del triangulo a dibujar

h1, h2, h3    indican laaltura especifica que se van a calcular para dividir 				algun punto a la mitad
w1, w2, w3    indican el ancho especifico que se van a calcular para dividir 				algun punto a la mitad

El programa muestra la cara del triangulo de Sierpinsky con un valor de n = 5 solicitado para este ejercicio, muestra una piramide en el centro y por motivos de tiempo no fue posible realizar la culminacion del mismo para poner las caras dentro de la piramide y girarlo.
